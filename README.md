# Smart Gate — Robotics Assignment

A vehicle detection system using ESP32, HC-SR04 ultrasonic sensor, servo motor, and RGB LED.

## Project Structure

| File | Description |
|------|-------------|
| `smart_gate.ino` | Part 1 — ESP32 C++ firmware |
| `blockly_open_gate.js` | Part 2 — Custom Blockly block definition & JS generator |
| `blockly_demo.html` | Part 2 — Live Blockly workspace demo |
| `smart_gate_dashboard.html` | Part 3 — Site Management Dashboard (HTML/CSS/JS) |

## How to Run
- **Part 1:** Open `smart_gate.ino` in Arduino IDE with ESP32Servo library installed
- **Part 2:** Open `blockly_demo.html` in any browser (internet required)
- **Part 3:** Open `smart_gate_dashboard.html` in any browser (no internet needed)
